🐾 AI Animal Detection

Learn Artificial Intelligence using Raspberry Pi, OpenCV & Python

📥 Download Code 📦 Requirements 📚 Libraries

🎯 Objective

Detect animals in real time using a Raspberry Pi and USB camera. When an animal is detected, GPIO18 activates a buzzer.

🛠 Required Components

  • 🍓 Raspberry Pi 4B
  • 📷 USB Camera
  • 🔊 Buzzer
  • 🔌 Jumper Wires
  • 🔋 5V Power Supply

🔌 Connections

ComponentPinConnection
BuzzerPositive (+)GPIO18
BuzzerNegative (-)GND
USB CameraUSBUSB Port

📋 Procedure

  1. Scan the QR code.
  2. Download the project files.
  3. Open Terminal.
  4. Create virtual environment: python3 -m venv mp-env
  5. Activate: source mp-env/bin/activate
  6. Install libraries: pip install -r requirements.txt
  7. Run: python3 Animal_Detection.py
  8. Point the camera towards an animal.
  9. The detected animal name appears on screen.
  10. The buzzer sounds when an animal is detected.

🤖 AI Working Flow

📷 Camera → 🧠 YOLO AI → 🐾 Animal Detected → 🔊 GPIO18 → 🚨 Buzzer
