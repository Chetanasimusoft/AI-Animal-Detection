🎯 Objective

This experiment demonstrates how Artificial Intelligence can detect animals in real time using a Raspberry Pi and a USB Camera. Whenever an animal is detected, the Raspberry Pi activates a buzzer to alert the user.

🛠 Required Components

  • 🍓 Raspberry Pi 4B
  • 📷 USB Camera
  • 🔊 Buzzer
  • 🔌 Jumper Wires
  • 🔋 5V Power Supply

🔌 Connections

Component Pin Connection
🔊 Buzzer Positive (+) GPIO 18
🔊 Buzzer Negative (−) GND
📷 USB Camera USB Port Raspberry Pi USB Port

📋 Procedure

  1. Scan the QR code provided in the manual.
  2. Download the project files.
  3. Open the Terminal on Raspberry Pi.
  4. Install all required libraries.
  5. Create a virtual environment.
  6. Activate the environment.
  7. Create a new Python file named Animal_Detection.py.
  8. Copy and paste the source code.
  9. Save the file.
  10. Run the Python program.
  11. The camera starts automatically.
  12. The AI continuously detects animals.
  13. When an animal is detected, the buzzer turns ON.

⚠ Troubleshooting

Problem Solution
Camera not detected Reconnect the USB camera and restart the Raspberry Pi.
Buzzer not working Check GPIO18 and GND wiring.
Module not found Install missing Python libraries.
Permission denied Activate the virtual environment before running the program.
